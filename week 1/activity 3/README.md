# Power Calculator Project

This project contains a simple Python program to calculate `power(x, y)`, meaning `x` raised to the power `y`.

## Files

- `power_calculator.py`: Python script that asks for `x` and `y` from the terminal and calculates `x^y`
- `screenshot.png`: Screenshot showing the current environment and sample program output

## How to Run

```bash
python power_calculator.py
```

After running the program, enter the values when prompted:

```text
Enter the value of x: 2
Enter the value of y: 5
```

Example output:

```text
x = 2
y = 5
power(x, y) = 32
```
